貯金箱トラッカー(フロントエンド)

貯金箱の中の500円、1円(取り出して使うことあり)の合計額と推移折線グラフを表示するシステム
フロントに+500円、-500円、+1円、-1円ボタン置いて、押したらAPI経由でnode+sqliteでデータ保存、APIで合計額と系列データを返す
